SIGNALS usage:

	-Refresh_Assets
		Reload all html/rss templates etc.
	-Plugins_Reload
		Reload all plugins.
	-Remove_UrlHook hook array
		Remove "hook" (absolute python path) from "array" (absolute python path).
	-Unload-Plugin plugin
		Remove plugin with name "plugin" (absolute python path)
